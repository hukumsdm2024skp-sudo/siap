☁️ Konfigurasi Cloud (Admin)Lokal

API Key & Bin ID dari JSONBin.io disimpan di sini. Setelah tersimpan, semua pegawai langsung bisa login tanpa perlu setup cloud.

💡 Cara kerja:
1. Admin isi API Key & Bin ID → klik Simpan
2. Sistem menyimpan konfigurasi ke cloud
3. Pegawai buka link aplikasi → langsung login tanpa setup apapun
4. Atau bagikan "Link Akses" ke pegawai agar otomatis terhubung
